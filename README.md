# Golang-Projetcs

Small projects developed as a way to better learn the Go language, following the guide in [this video](https://youtu.be/jFfo23yIWac)

## Useful Commands
```bash
#Create Module file
go mod init <name>

#Run a file
go run <file>/.
 
#Build a project
go build

#Get dependencies
go get <dependency>

```
